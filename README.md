# LabExp2
Repositório para o segundo trabalho do Laboratório de Experimentação de Software.

## Enunciado
Analisar aspectos da qualidade de repositórios desenvolvidos na linguagem Java, correlacionando-os com características do seu processo de desenvolvimento, sob a perspectiva de métricas de produto calculadas através da ferramenta CK.
